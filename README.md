# 💜 Tech em Foco

Projeto desenvolvido utilizando HTML, CSS e JavaScript.

## 📚 Sobre

O objetivo deste projeto é apresentar um blog simples sobre tecnologia com uma interface moderna e um sistema de interação por curtidas.

## 🚀 Funcionalidades

- Dois posts sobre tecnologia;
- Sistema de ❤️ Curtidas;
- Sistema de 👍 Joinhas;
- Contadores salvos no navegador utilizando LocalStorage;
- Layout moderno e responsivo.

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript
